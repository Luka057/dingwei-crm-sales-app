# 接口文档

基础路径：`/api/v1`

## 1. 登录

`POST /auth/login`

请求：

```json
{
  "username": "zhangwei",
  "password": "123456"
}
```

响应：

```json
{
  "token": "jwt-token",
  "user": {"id": "u1", "name": "张伟", "role": "sales"}
}
```

## 2. 获取我的日历

`GET /plans/calendar?year=2026&month=5`

响应：

```json
{
  "year": 2026,
  "month": 5,
  "days": [
    {
      "date": "2026-05-14",
      "items": [
        {"id": "p1", "type": "visit", "title": "宏远服饰", "time": "09:30", "status": "pending"}
      ]
    }
  ]
}
```

## 3. 新增日程

`POST /plans`

请求：

```json
{
  "type": "visit",
  "customerId": "c1",
  "title": "宏远服饰",
  "scheduledAt": "2026-05-14T09:30:00+08:00",
  "content": "确认宽度调整样板反馈"
}
```

## 4. 修改日程

`PUT /plans/{id}`

## 5. 删除日程

`DELETE /plans/{id}`

## 6. 客户列表

`GET /customers?keyword=宏远&level=A&overdue=true`

## 7. 客户详情

`GET /customers/{id}`

## 8. 新增拜访记录

`POST /visit-records`

请求：

```json
{
  "customerId": "c1",
  "visitAt": "2026-05-14T09:30:00+08:00",
  "method": "phone",
  "intention": "likely_order",
  "targetPerson": "李总",
  "targetTitle": "总经理",
  "content": "客户认可手感，担心颜色批次稳定性。",
  "nextFollowAt": "2026-05-20T10:00:00+08:00"
}
```

## 9. 周报

- `GET /weekly-reports`
- `GET /weekly-reports/{id}`
- `POST /weekly-reports`
- `PUT /weekly-reports/{id}`
- `POST /weekly-reports/generate-ai-draft`

## 10. AI 对话

`POST /ai/chat`

请求：

```json
{
  "message": "宏远最近订单和拜访情况？"
}
```

## 11. AI 找板

`POST /ai/board-search`

请求：

```json
{
  "customerScene": "宏远服饰 运动服织带",
  "imageIds": ["file_1"],
  "width": "20mm",
  "tension": "中高",
  "ribbonType": "提花织带",
  "requirements": "黑色，手感偏软，颜色批次稳定"
}
```

响应：

```json
{
  "matches": [
    {
      "sampleNo": "HY-20-BK-软手感",
      "score": 0.92,
      "reason": "宽度、颜色、手感要求相似，关联宏远历史样板。"
    }
  ]
}
```
