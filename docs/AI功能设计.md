# AI 功能设计

## 1. AI 摘要

输入：拜访内容自由文本。

输出：

- 客户核心反馈
- 需求点
- 风险点
- 下次跟进建议

## 2. AI 查询

支持问题：

- 某客户上次说了什么？
- 本周未拜访的 A 类客户有哪些？
- 哪些样板还没确认？
- 本月拜访排名如何？
- 某客户订单和样板转化情况？

## 3. AI 周报

输入：

- 本周计划
- 本周拜访记录
- 样板状态变化
- 未完成事项

输出固定结构：

- 本周工作总结
- 下周工作计划
- 备注事项
- 附件建议

## 4. AI 找板

输入：

- 图片
- 客户/场景
- 宽度
- 开力度
- 织带类型
- 颜色/要求

输出：

- 匹配样板编号
- 匹配度
- 匹配原因
- 关联客户历史记录
- 推荐下一步动作

## 5. Function Calling 工具

- `search_customers`
- `get_customer_detail`
- `search_visit_records`
- `search_samples`
- `search_orders`
- `generate_weekly_report`
- `search_board_samples`

## 6. 安全要求

- API Key 只存后端。
- 前端只调用业务后端。
- AI 查询必须带用户权限过滤。
- 销售只能查询自己负责客户。
