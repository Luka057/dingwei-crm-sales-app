# 数据字典

## users 用户表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 用户 ID |
| username | varchar | 登录账号 |
| password_hash | varchar | 密码哈希 |
| name | varchar | 姓名 |
| role | enum | boss / manager / sales |
| manager_id | uuid | 所属主管 |
| status | enum | active / disabled / resigned |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## customers 客户表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 客户 ID |
| name | varchar | 客户名称 |
| short_name | varchar | 简称 |
| level | enum | A / B / C |
| owner_id | uuid | 负责销售 |
| contact_name | varchar | 联系人 |
| contact_title | varchar | 联系人职位 |
| phone | varchar | 电话 |
| address | varchar | 地址 |
| last_visit_at | datetime | 最近拜访时间 |
| status | enum | active / inactive |
| created_at | datetime | 创建时间 |

## visit_records 拜访记录表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 拜访记录 ID |
| customer_id | uuid | 客户 ID |
| salesperson_id | uuid | 销售 ID |
| visit_at | datetime | 拜访时间 |
| method | enum | offline / phone / wechat |
| intention | enum | good / likely_order / wait / none |
| target_person | varchar | 拜访对象 |
| target_title | varchar | 对象职位 |
| content | text | 拜访内容 |
| ai_summary | text | AI 摘要 |
| next_follow_at | datetime | 下次跟进 |
| created_at | datetime | 创建时间 |

## plans 工作计划表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 计划 ID |
| salesperson_id | uuid | 销售 ID |
| customer_id | uuid | 可为空 |
| title | varchar | 标题 |
| type | enum | visit / custom |
| scheduled_at | datetime | 计划时间 |
| content | text | 内容 |
| status | enum | pending / done / cancelled |

## samples 样板表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 样板 ID |
| customer_id | uuid | 客户 ID |
| salesperson_id | uuid | 销售 ID |
| sample_no | varchar | 样板编号 |
| status | enum | 起样 / 中板 / 确认 / 已转订单 / 已放弃 |
| width | varchar | 宽度 |
| tension | varchar | 开力度 |
| ribbon_type | varchar | 织带类型 |
| color | varchar | 颜色 |
| image_url | varchar | 图片 |
| notes | text | 备注 |

## weekly_reports 周报表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | uuid | 周报 ID |
| salesperson_id | uuid | 销售 ID |
| week_start | date | 周开始日期 |
| summary | text | 本周工作总结 |
| next_plan | text | 下周工作计划 |
| notes | text | 备注事项 |
| attachments | json | 附件 |
| status | enum | draft / submitted |
