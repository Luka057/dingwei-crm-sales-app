# 部署说明

## 1. 部署目标

支持内网环境 Docker Compose 一键启动。

## 2. 服务组成

- `frontend`：前端静态资源
- `backend`：业务 API
- `postgres`：数据库
- `redis`：缓存
- `nginx`：反向代理

## 3. 环境变量

```env
DATABASE_URL=postgresql://crm:crm_password@postgres:5432/dingwei_crm
REDIS_URL=redis://redis:6379
JWT_SECRET=change_me
DEEPSEEK_API_KEY=change_me
```

## 4. 启动

```bash
docker compose up -d
```

## 5. 初始化

```bash
docker compose exec backend npm run db:migrate
docker compose exec backend npm run db:seed
```

## 6. 访问

- 前端：`http://内网IP`
- 后端：`http://内网IP/api/v1`
