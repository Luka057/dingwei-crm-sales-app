# 原型交互说明

## 1. 原型位置

- 直接打开：`frontend/prototype/crm-sales-timepage-prototype.html`
- 前端工程访问：`/prototype/crm-sales-timepage-prototype.html`

## 2. 页面结构

底部导航固定四个页面：

| 页面 | 说明 |
|---|---|
| 日历 | 拜访计划、自定义事项、日历浮窗、日期时间线 |
| 客户 | 我的客户、筛选、客户详情、拜访时间线 |
| 周报 | 历史周报、固定格式录入、AI 草稿 |
| AI | 客户/拜访/样板/订单问答，AI 找板 |

## 3. 日历页交互

### 日程优先

- 默认显示每日时间线。
- 左上角菜单按钮打开完整月历浮窗。
- 点击月历日期后回到对应日期时间线。
- 上下滚动可连续浏览前后日期。
- 每一天标题右侧有 `+ 拜访`、`+ 计划`。
- 右下角全局 `+` 保留，默认新增当前选择日期的日程。

### 月历优先

- 月历直接显示在日程页上方。
- 点击日期后下方显示当天时间线。

## 4. 新增与编辑事项

事项类型只保留：

- 拜访
- 自定义内容

新增后写入 demo 的内存数据，页面内可继续查看、编辑、完成、删除。原型刷新后不会持久化 demo 事项，这是当前单 HTML 原型限制。

## 5. 主题能力

显示模式：

- 浅色模式
- 深色模式

日程主题：

- 日程优先
- 月历优先

整体配色：

- 暖纸原色
- 清爽蓝调
- 松绿色调
- 淡紫色调
- 石墨灰调

深色模式下五套色调都有各自深色底，不是单一黑色。

## 6. 选中态

小模块选中态采用半透明毛玻璃：

- 客户筛选 chip
- 表单分段按钮
- 主题选项
- 配色选项
- AI 抽屉当前项

主操作按钮仍保持明确实底，例如：搜索、保存、提交、查询。

## 7. AI 找板

入口：AI 页右上角 `+`。

输入：

- 图片
- 客户/场景
- 宽度
- 开力度
- 织带类型
- 要求描述

输出：demo 匹配历史样板编号、匹配度、原因。
