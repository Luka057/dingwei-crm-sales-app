## 说明

- `prototype/` 保留单文件高保真原型，适合快速改 UI 和交互。
- `public/prototype/` 是同一份原型，用于前端工程部署后访问。
- `src/` 是后续工程化 React 入口。
- `backend/src/main.ts` 是 mock API，便于前端先接接口。
- `docs/agent/AGENT_HANDOFF.md` 是下一位 agent 衔接时最应该先读的文件。
